# Non-Invasive Cardiac Output Monitor (NICOM) UI

This project demonstrates a simulated Non-Invasive Cardiac Output Monitor (NICOM) user interface,
designed according to medical device standards and clinical reference limits.

## Device Description
The NICOM is a hemodynamic monitoring device used to assess cardiac performance without invasive procedures.
It displays numeric parameters such as cardiac output, stroke volume, and heart rate.

## Parameters Displayed
- Cardiac Output (CO) – L/min
- Stroke Volume (SV) – mL/beat
- Heart Rate (HR) – bpm

## Features
- Real-time parameter simulation using sliders
- Visual status indicators (Normal / Out of Range)
- Clinical warning messages when parameters exceed safe limits

## Standards Applied
- IEC 60601-1-8: Alarm systems for medical electrical equipment
- IEC 62366-1: Usability engineering for medical devices
- Clinical hemodynamic reference ranges for physiological limits
